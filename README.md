### Node.js 설치
  https://nodejs.org/ko/
  * LTS로 다운로드 받기

### Node-RED 명령어 설치
  (맥/우분투)sudo npm install -g node-red
    * 안 될 경우: sudo apt install npm 입력 후 설치
  (윈도우)npm install -g node-red 
  
  
## Node-RED 실행
  node-red
  
## Node-RED 접속
  크롬에서 "localhost:1880" 입력하여 접속
